The PR Review Bot is an automated code-review assistant built with Node.js and powered by the Gemini API. It analyzes pull-request diffs, understands code changes, and generates meaningful review feedback to support developers during the review process. The bot identifies potential issues, suggests improvements, highlights code-quality concerns, and produces concise summaries to help teams reduce review time and maintain consistent coding standards. It can be triggered through GitHub webhooks or executed manually for on-demand reviews, making it a practical tool for fast-paced development workflows.



Features:-
Reviews PR diffs and provides AI-generated feedback
Suggests code improvements and best practices
Summarizes changes for faster review cycles
Supports GitHub webhooks / CLI trigger
Lightweight and easy to deploy

Tech Stack
Node.js
Gemini API
GitHub API / Webhooks

Prerequisites
Node.js (LTS)
Gemini API Key
GitHub Personal Access Token (if using GH integration)
